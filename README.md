# AI Wafer Defect Detection System

AI-powered semiconductor wafer defect detection and analytics system built using Python, OpenCV, TensorFlow, and Streamlit.

## Project Overview

This project detects semiconductor wafer defects using computer vision and deep learning techniques.
The system analyzes wafer images, identifies defects automatically, and provides real-time analytics through an interactive dashboard.

## Features

* Automated wafer defect detection
* Image preprocessing and edge detection
* AI-based defect classification
* Real-time analytics dashboard
* Defect heatmap visualization
* Interactive Streamlit interface
* Semiconductor quality monitoring

## Technologies Used

* Python
* OpenCV
* TensorFlow
* Streamlit
* Pandas
* NumPy
* Matplotlib

## How to Run

1. Clone the repository

git clone https://github.com/Pratiksha80/AI-Wafer-Defect-Detection-System.git

2. Install required libraries

pip install -r requirements.txt

3. Run the project

streamlit run app.py
